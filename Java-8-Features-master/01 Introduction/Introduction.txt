Java 8 new Features: -
----------------------
1. Lambda expression
2. Functional interface : used to call lambda expression
3. Default methods in interfaces
4. Static methods in interfaces
5. Predicate
6. Function
7. Consumer, Supplier
8. Method reference & constructor reference by double colon(::) operator
9. Stream API : for more readable code
10. Date & time API : introduced by joda.org also called as JODA API


from java 1.8 v onwards we can create concrete methods (default, static method) in interface.

Predicate, Function, Consumer and Supplier all these are Pre-defined functional interfaces.
